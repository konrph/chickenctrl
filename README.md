# chickeCTRL
***
Smart controller for a chicken barn


<br/>

###MQTT Commands

|      Topic      | Values | <div style="width:100px">Action</div> |
|:---------------:|:------:|:-------------------------------------:|
|  /control/door  |   0    |               Stop Door               |
|                 |   1    |               Open Door               |
|                 |   2    |              Close Door               |
| /control/feeder |   0    |              Stop Feeder              |
|                 |   1    |             Start Feeder              |
| /control/light  |   0    |               Light off               |
|                 |   1    |               Light on                |
